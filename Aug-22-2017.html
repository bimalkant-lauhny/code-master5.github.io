GSoC 2017 - Project Summary

August 22, 2017 / by Bimalkant Lauhny

Image source from opensource.googleblog.com

On May 5, 2017 I got selected as a GSoC 2017 participant under openSUSE . I contributed to the Open Source project Jangouts under the program during from May 29, 2017 to August 29, 2017. This post summarises my contributions to the project throughout this GSoC.


Image source from 101.opensuse.org


What I've done?


Following is the list of my contributions before GSoC official coding period.

  • Solved a bug which was stopping Jangouts client from reading config values.
    • Status: Complete and Merged
    • Issue
    • Pullrequest
  • Added support for changing username during a conference.
    • Status: Complete and Merged
    • Issue
    • Pullrequest
  • Added support for PIN protected rooms.
    • Status: Complete and Merged
    • Issue
    • Pullrequest
  • Added cross-browser Fullscreen video support in Jangouts.
    • Status: Complete and Unmerged
    • Issue
    • Pullrequest


The title of my GSoC project is - Intergrate callstats.io into Jangouts . Following is the list of my contributions during GSoC official coding period.


Part 1: Intergration of callstats.io service into Jangouts.

  • Added documentation explaining the interaction between components in Jangouts
    • Status: Complete and Merged
    • Pullrequest
  • Added event emitters into Jangouts.
    • Status: Complete and Unmerged
    • Pullrequest
  • Added callstats.io service support into Jangouts.
    • Status: Complete and Unmerged
    • Pullrequest

Part 2: Intergration of callstats.io service into Janus-gateway.

  • Node based callstats.io integration using callstats REST API
    • Status: Complete
    • Prototype
  • C based callstats.io integration using callstats REST API.
    • Status: Pending
    • Pullrequest
